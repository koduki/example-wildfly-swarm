Example for wildfly-swarm
==================================

This [Swarm](http://wildfly.org/swarm/) example uses JPA, CDI and JAX-RS.


Run
----------------------------------

```
$ mvn package
$ java -jar target\example-wilfly-swarm-0.1-swarm.jar
```

or

IDE -> Run

Use
----------------------------------

```
http://localhost:8080/employees
```
